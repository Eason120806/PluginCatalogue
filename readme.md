# PluginCatalogue

[中文](readme_cn.md) | **English**

A repository for listing [MCDReforged](https://github.com/Fallen-Breath/MCDReforged) plugins.

If you have a good plugin, please submit a pull request and make sure you sort the tables by **1.x Support -> Initials**.

## Format and Summary

### Format

| Name | Maintainer | Function | 1.x Support |
| - | - | - | - |
| Example 1 | Nobody | This is a plugin that supports **MCDR 1.x** | :large_blue_circle: |
| Example 2 | Nobody | This is a plugin that only supports **MCDR 0.x** | :red_circle: |
| ~~Example 3~~ | Nobody | This is a **invalid** plugin | :question: |
