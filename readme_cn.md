# MCDR插件仓库

**中文** | [English](readme.md)

一个用于列出 [MCDReforged](https://github.com/Fallen-Breath/MCDReforged) 插件的仓库。

有什么好插件的话请提交 PR，确保将表格按照 **兼容 1.x -> 插件首字母** 排序。

## 格式和目录

### 格式

| 名称 | 维护者 | 功能 | 兼容 1.x |
| - | - | - | - |
| Example 1 | Nobody | 这是一个支持 **MCDR 1.x** 的插件 | :large_blue_circle: |
| Example 2 | Nobody | 这是一个只支持 **MCDR 0.x** 的插件 | :red_circle: |
| ~~Example 3~~ | Nobody | 这是一个 **无效** 插件 | :question: |
